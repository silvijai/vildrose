# Word Sizes

The project starts from Trit, and build upon it with other types.

Planned sizes include Word9 (also referred to as tryte sometimes), and Word27 (and later on, Word54 and maybe Word108).

Word will be a first class citizen in the ISA and CPU implementation. Word27 is planned to be the size for the CPU, with it having a Word27 amount of registers and. I will likely end up referring to this as a 27-trit CPU architecture, analogous to 32-bit CPU architecture.
